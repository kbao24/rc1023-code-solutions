# react-jsx-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What is JSX?
  JSX is a syntax extension for JS that lets you write HTML markup in a JS file
- How does React use JSX to render components?
  React allows developers to define the structure of user interface in JS files.
